## Decisiones tomadas
- `$timestamps = false` + `created_at` manual (la auditoría no se modifica)
- Trait protege contra recursión: no loggea si no hay `auth()->user()`
- `updated` filtra cambios vacíos y excluye `updated_at` del diff
- `restored` solo se registra si el modelo usa `SoftDeletes`
- Para activar la auditoría en un modelo: agregar `use App\Traits\Auditable;` en el modelo
